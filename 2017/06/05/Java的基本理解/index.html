Java的基本理解

Java是一种强类型语言，所谓强类型语言从两方面解释:
1.所有的变量必须先声明后使用
2.指定类型的变量只能接受与之类型匹配的值
这意味着每个变量或表达式在编译时期就明确了其类型

Java数据类型分类
基本数据类型:
{
(n个字节 一个字节占8位)
整数:[ byte (1)、short (2)、int (4)、long (8) ]
字符:[ char (2) ]
浮点:[ float (4)、double (8) ]
布尔:[ boolean ]
}
