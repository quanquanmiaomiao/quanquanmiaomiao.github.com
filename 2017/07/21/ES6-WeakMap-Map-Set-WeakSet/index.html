ES6-WeakMap/Map/Set/WeakSet

Map

Map是ES6新增的有序键值对集合,键值对(key,value)可以使任何类型的元素。

WeakMap

你可以把WeakMap看作是Map的子集,它拥有和Map类似的set(key,value)、get(key)、has(key)、delete(key)、和clear()等方法，但没有size()属性,也没有任何与迭代相关的方法,如:no .entries(), no .keys(), no .values(), no .forEach(), no .clear()。

在WeakMap中key只能是非空的对象,WeakMap仅保持对key的弱引用,可以避免内存泄漏。

Set

Set是ES6新增的有序列表集合，它不包含重复项。

采用Object方式实现没有重复项存在的问题

上述代码中最终o的结果为{2:2,[object,object]:4}, why?
在object中会对key进行toString()操作,导致key对应的数据被覆盖。

数组可以存放任何类型的数据，不过数据去重需要开发者手动实现。

Set可以有效的解决上面存在的问题。
Set可以通过add(item)方法想Set添加任意类型的元素，如果已经添加过则自动忽略。

注意:Set调用add、has、delete等方法对key进行比较不会做类型转换，可以使用 === 进行比较，但是也存在特殊情况:
NaN 只能添加一次 虽然 NaN === NaN 为false
-0或0或+0 可以同时存在,因为符号不一样虽然 -0===0 -0===+0 true

WeakSet

WeakSet和Set就像是WeakMap与Map的关系一样。WeakSet中的value只能是非空的对象且不能使用迭代器。
