CreateReactNativeApp-Expo

###认识###
CreateReactNative是一个移动端APP开发套件，它可以让我们采用JavaScript进行开发高质量的iOS和Android应用。通过CreateReactNative开发React Native的时候，我们不需要打开xcode和android stduio甚至也许不要写任何原生代码。它是Facebook和Expo合作的产物,它支持ReactNative提供的所有组件以及大部分的API,并支持Expo提供的API。
但是当我们不得不用到Native Code时怎么办？
Expo提供了对设备系统功能的访问比如(相机、手机联系人、本地存储以及其他硬件的访问)。
开发过程中引入需要使用react-native link的依赖时我们又怎么办？
我们可以使用ejecting, 提供运行 npm run eject 得到一个与react-native init类似的项目，此时你就需要XCode或者Android Studio。
